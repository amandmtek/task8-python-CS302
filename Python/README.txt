Описание
Quest Master — это настольное приложение, разработанное для помощи пользователю в организации заданий и ведении учета их прогресса.

Генератор Квестов - Создание, редактирование и автосохранение квестов в локальной базе данных.
Экспорт - Сохранение квестов в форматах PDF, DOCX с использованием шаблонов.
Редактор Карт - Инструмент для быстрого наброска карт, путей и ключевых локаций .
Отслеживание опыта и уровня пользователя за активность в приложении.

---

Инструкция по Установке и Запуску

1. Требования
Для работы программы необходим командная строка и версия Python 3.8+.

2. Введите в командную строку: pip install -r requirements.txt

3. Запуск приложения происходит через файл main.py
